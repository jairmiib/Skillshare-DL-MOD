# Skillshare-DL-MOD
K-E-N-W-A-Y Skillshare-DL MOD para descargar localmente cursos de Skillshare.
Todos los créditos para K-E-N-W-A-Y y el autor original del código.

# Uso
Usa el siguiente comando para descargar cualquier curso
`python dl.py <URL del curso>`

Por ejemplo
`python dl.py https://www.skillshare.com/classes/Hand-Lettering-Essentials-for-Beginners/389616295`
